「毎日投稿ノルマ」ではなく在庫。反応が良かった切り口は深掘りして追加を作る。
機能紹介だけでなく、リリース後の学び・アップデート報告を混ぜると宣伝臭が薄まる。

## ストック1: 猫の鳴き声通知

添付: `feature-catsound.gif`

```text
AgentManagerの通知は猫の鳴き声です。

通知センターは使いません。バナーが積み上がらないし、作業の邪魔をしない。でも「にゃっ」と鳴いたらClaude Codeが呼んでる。

もちろんミュートもできます🐈
```

## ストック2: 昼夜の切り替え

添付: `screenshot-night.gif`

```text
AgentManagerの猫部屋、夜になると部屋も夜になります。

起きてる猫は目が光る。深夜にClaude Codeを回してると、猫も一緒に夜更かししてる感じになってちょっといい。
```

## ストック3: ⌥Space トグル

添付: `feature-keyboard.gif`

```text
AgentManagerは⌥Spaceでパッと出してパッと消せます(キーは変更可)。

↑↓でセッション選択、→でそのターミナルへジャンプ、Escで閉じる。マウスなしで完結します。
```

## ストック4: シンプルモード

添付: `feature-simple.png`

```text
「猫はかわいいけど仕事中は集中したい」という人のために、装飾を全部消したシンプルモードがあります。

セッション状態のリストだけが淡々と並ぶ。どっちのモードでも中身は同じです。
```

## ストック5: メニューバー

添付: `feature-menubar.png`

```text
ウィンドウを閉じていても、メニューバーに状態別の件数が色付きで出ています。

琥珀=確認待ち、緑=完了、青=処理中。琥珀が点いたら⌥Space。それだけ覚えれば使えます。
```

## ストック6: IDE対応

添付: `feature-ide.png`

```text
AgentManagerはターミナルだけじゃなく、VS Code / Cursor / Android Studioの統合ターミナルで動くClaude Codeセッションも検知します。

エディタ内で回してるセッションの確認待ちにも気づけます。
```
