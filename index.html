macOS 13 Ventura 以降 · Apple Silicon / Intel 対応

主な機能

🟡

確認待ちを即検知

ツール許可・プラン承認・質問回答など、Claude が止まっている理由をリアルタイムで表示。

🖱

1クリックでターミナルへ

行をクリックすると対応する iTerm2 / Terminal / Warp のペインへ即ジャンプ。

⌥Space

グローバルホットキー

⌥Space でどのアプリからでも一瞬でパネルを呼び出し、消す。

🐱

セッション連動の猫

部屋の床を走る猫がセッション数と状態に連動。時刻で昼・夕・夜の景色が変わります。

🔄

自動アップデート

Sparkle によるアプリ内自動更新。起動時に新バージョンがあれば通知します。

🔒

ライセンス認証

Freemius によるサブスクリプション管理。無料トライアル後に購入して有効化するだけ。

インストール手順

  1. 1
    購入 / トライアル開始 「購入する」ボタンから Freemius チェックアウトへ。メールでライセンスキーが届きます。
  2. 2
    DMG をダウンロード 「DMG をダウンロード」ボタンからインストーラーを取得します。
  3. 3
    Applications にドラッグ DMG を開き AgentManager.app を Applications フォルダへドラッグ。
  4. 4
    ライセンスキーを入力 アプリを起動して 設定 → ライセンス にキーを貼り付けて「有効化」。

動作環境

macOS 13 Ventura 以降 Apple Silicon / Intel Claude Code（claude CLI） iTerm2 / Terminal / Warp
